Proposta de projeto

Nos foi proposto pelo professor de sala de aula Cássio Gimenez que desenvolvêssemos em equipe um aplicativo Todo (Lista de tarefas) utilizando React Native. Os critérios estabelecidos incluíam a implementação de botões para editar, excluir e adicionar tarefas, capacidade de selecionar e exibir imagens, criação de uma página de login e a integração de um sensor para acessar um site.

Código React Native

Função para adicionar uma nova tarefa

Função para editar uma tarefa existente

Função para excluir uma tarefa existente e Função para selecionar uma imagem da galeria

Componete de Login

Efeito para detectar agitação e abrir o site
